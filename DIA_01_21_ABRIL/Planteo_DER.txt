Ejercicio
Una mueblería necesita la implementación de una base de datos para controlar las ventas que realiza por día, 
el stock de sus artículos (productos) y la lista de sus clientes que realizan las compras.

Se necesita plantear:
¿Cuáles serían las entidades de este sistema?
Las entidades serian: Productos, Clientes, Ventas
¿Qué atributos se determinarán para cada entidad? (Considerar los que se crean necesarios)
Los atributos para Productos:
  -idProducto
  -nombre
  - Precio
  - idVenta
  
Los atributos para Clientes:
  -idCliente
  -nombre
  -DNI
  -idVenta

Los atributos para Ventas:
  -idVenta
  -fechaVenta
  -TotalVenta
  -idProducto
  -idCliente

¿Cómo se conformarán las relaciones entre entidades? 


¿Cuáles serían las cardinalidades?
Un cliente podra comprar varios productos 
Una venta puede tener varios productos
Un producto puede estar en varias ventas

Realizar un DER para modelar el escenario planteado.
